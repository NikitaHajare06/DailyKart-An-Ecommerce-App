# E-Commerce Application

This is a starter E-Commerce application scaffold with a backend API and a React frontend.

## Structure

- `backend/` - Node.js Express API for product data and order endpoints
- `frontend/` - React application for browsing products and cart flow
- `.github/copilot-instructions.md` - workspace instructions for Copilot

## Quick Start

1. Open the workspace in VS Code.
2. Install dependencies in both folders:
   - `cd backend && npm install`
   - `cd frontend && npm install`
3. Start the backend and frontend separately:
   - `cd backend && npm start`
   - `cd frontend && npm start`

## Notes

This scaffold is intended as a baseline. Replace placeholder product data and extend payment, authentication, and inventory features as needed.
